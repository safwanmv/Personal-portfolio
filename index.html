AI ChatBot

Dubai Trip Assistant is an AI-powered travel chatbot designed to help users plan their perfect trip to Dubai with ease. Built using Python, Streamlit, and the OpenAI API, this smart assistant provides personalized recommendations for attractions, food spots, and hotels, and even generates day-wise travel itineraries based on user preferences. With a clean and modular code structure, the project is easy to customize and deploy on Streamlit Cloud. Key technologies include Python 3, Streamlit for the frontend interface, OpenAI GPT for AI responses, dotenv for secure key management, and Git for version control. This chatbot offers a seamless and interactive way to explore Dubai like never before.
